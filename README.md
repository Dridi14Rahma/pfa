\# Skill Tracker



Application de suivi et de gestion des compétences techniques.



\## Stack Technique



| Composant | Technologie |

|-----------|-------------|

| Backend | Spring Boot 3.3.5 (Java 17) |

| Frontend | Angular 17 |

| Base de données | MongoDB 6 |

| Serveur Web | Nginx |

| Conteneurisation | Docker |

| Orchestration | Kubernetes |

| CI/CD | GitHub Actions |



\## Prérequis



\- Docker Desktop

\- Node.js 20 (pour développement local)

\- Java 17 (pour développement local)



\## Démarrage rapide avec Docker



```bash

\# Cloner le projet

git clone https://github.com/TON\_USER/skill-tracker.git

cd skill-tracker



\# Build et démarrer les conteneurs

docker-compose up --build



\# Arrêter les conteneurs

docker-compose down
